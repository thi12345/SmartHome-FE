# SmartHome-FE
# How to run this project
  - Run 'git clone https://github.com/thi12345/SmartHome-FE.git' to clone this project
  - After, run 'git checkout master' to switch **master** branch
  - Run 'install' to install all package in this project
  - Finally, run 'ng serve' to run the project on localhost:4200
